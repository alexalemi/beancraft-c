beancraft / counter-machine playground

A counter machine — just increment and decrement-or-branch — running in your browser via the C interpreter compiled to WebAssembly. Pick an example or write your own; the optimizer (-O) is on.

Screen (live, 256×192 — clicks & movement feed mouse/*)

Built with make wasm (Emscripten, Asyncify). The QBE → native compiler in beancraft can't run client-side, so this is the interpreter only. Programs that draw via screen/* animate live on the canvas above — every inc screen/flush paints a frame and yields — and while a run is live, your keyboard feeds kbd/event, the canvas feeds mouse/*, and the input box feeds con/read. Runs end at halt, the step cap, or the Stop button (the run yields between work chunks, so even a tight compute loop stays interruptible). Source: github.com/alexalemi/beancraft-c.
